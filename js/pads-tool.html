PADS ASCII Parser Tool

Parses PADS netlist files (*PADS-PCB*), PADS Layout ASCII exports (!PADS-POWERPCB-...) and PADS Logic schematic exports (*PADS-LOGIC*). Runs entirely in your browser — no install, no network, nothing leaves this page.

Drop a .asc / .pads file here or click to browse

“Copy Layout Parts” puts tab-separated columns on the clipboard — Ref Des, Part Type, Decal/Footprint, Side, X, Y, Rotation — ready to paste into an Excel parts sheet. “Copy Schematic Parts” copies Ref Des, Part Type, PCB Decal, EDU Part Number, FLIGHT Part Number (part numbers filled from schematic attributes when present). The full attribute set is in the Parts tab and the Parts CSV download.
