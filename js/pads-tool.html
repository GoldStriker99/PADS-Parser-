PADS ASCII Parser Tool

Parses PADS netlist files (*PADS-PCB*) and PADS Layout ASCII exports (!PADS-POWERPCB-...). Runs entirely in your browser — no install, no network, nothing leaves this page.

Drop a .asc / .pads file here or click to browse

“Copy Layout Parts” puts tab-separated columns on the clipboard — Ref Des, Part Type, Decal/Footprint, Side, X, Y, Rotation — ready to paste into an Excel parts sheet. “Copy Schematic Parts” copies Ref Des, Part Type only.
